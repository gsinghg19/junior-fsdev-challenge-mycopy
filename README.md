### Set-up

To run the app, install dependencies viathe command:

```
npm install
```

Then create an .env file in the root folder of the app. Enter your Google firebase api key here, as described in the '.env.example.js' file.

## Commands

To run the app:

```
npm start
```

To run the test:

```
npm test
```
